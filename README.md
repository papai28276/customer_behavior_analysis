📊Customer Behaviour Analysis
This project analyzes customer shopping data using Python, SQL, and Power BI.

📁 Files
customer_behavior.ipynb → Data cleaning and analysis using Python
customer_behaviour_sql.sql → SQL queries for insights
customer_shopping_behavior.csv → Dataset
customer_behaviour_dashboard.pbix → Power BI dashboard

🔍 What I Did
Cleaned and analyzed data using Python
Used SQL to find important insights
Created a dashboard in Power BI to visualize data

📊 Key Insights
Customers with discounts sometimes spend more
Subscription affects customer spending
Some products have higher ratings than others
Shipping type impacts purchase behavior

🛠️ Tools Used
Python (Pandas, NumPy)
SQL
Power BI

▶️ How to Use
Open the Python notebook for analysis
Run SQL queries in your database
Open the Power BI file to see the dashboard

⭐ Purpose
This project helps understand customer behavior and improve business decisions.
